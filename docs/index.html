Terminal Souls

Terminal Souls é um TUI-ARPG (RPG de Ação com Interface de Texto) inspirado em Dark Souls e Path of Exile. Experiencie uma interface de terminal minimalista que serve como uma janela para sistemas matemáticos e lógicos profundos.

Recursos

  • Sinergia Estratégica (Min-Maxing): Encadeie efeitos, tags e gatilhos de itens para dano máximo.
  • Combate Punitivo (Risco vs. Recompensa): Combate deliberado por turnos onde o gerenciamento de recursos (Vigor/Mana) é fundamental.
  • Intelecto como Arma: Resolva quebra-cabeças lógicos e matemáticos gerados proceduralmente para progredir e desbloquear itens de alto nível.
  • Estética TUI Imersiva: Hierarquia visual limpa, nostálgica e altamente informativa usando Chalk e Blessed.

Ciclo de Jogo

Explore mapas 2D gerados proceduralmente, engaje em combates por turnos e resolução de quebra-cabeças, colete itens e melhore seu equipamento para progredir através de níveis cada vez mais difíceis.

Arquitetura Técnica

  • Tempo de Execução (Runtime): Node.js (v18+)
  • Interface: blessed (Layout e Entrada), chalk (Cores/Estética)
  • Lógica de Dados: uuid (IDs de itens únicos), mathjs (Geração de Quebra-cabeças)

Status

Atualmente em Alpha 0.0.1. O desenvolvimento está em andamento para implementar persistência, progressão de personagens, hubs sociais (Nexus) e sinergias de itens complexas.
