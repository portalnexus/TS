Terminal Souls

Terminal Souls is a TUI-ARPG (Text User Interface Action RPG) inspired by Dark Souls and Path of Exile. Experience a minimalist terminal interface that serves as a window to deep mathematical and logical systems.

Features

  • Strategic Synergy (Min-Maxing): Chain effects, tags, and item triggers for maximum damage.
  • Punishing Combat (Risk vs. Reward): Deliberate turn-based combat where resource management (Stamina/Mana) is key.
  • Intellect as a Weapon: Solve logical and mathematical puzzles generated procedurally to progress and unlock high-tier loot.
  • Immersive TUI Aesthetics: Clean, nostalgic, and highly informative visual hierarchy using Chalk and Blessed.

Gameplay Loop

Explore procedurally generated 2D maps, engage in turn-based combat and puzzle-solving, gather loot, and upgrade your equipment to progress through increasingly difficult levels.

Technical Architecture

  • Runtime: Node.js (v18+)
  • Interface: blessed (Layout and Input), chalk (Colors/Aesthetics)
  • Data Logic: uuid (Unique item IDs), mathjs (Puzzle Generation)

Status

Currently in Alpha 0.0.1. Development is ongoing to implement persistence, character progression, social hubs (Nexus), and complex item synergies.
