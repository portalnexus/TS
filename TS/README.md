# Terminal Souls: The Echoes of Reason

**Terminal Souls** é um TUI-ARPG (Text User Interface Action RPG) inspirado em Dark Souls e Path of Exile, agora com um foco profundo na **História da Ciência e Matemática**.

## 🌌 O Conceito
No mundo do Exílio, o conhecimento é a única arma real. As fendas não são apenas masmorras, mas ecos de mentes brilhantes do passado. O jogador deve usar o intelecto para resolver puzzles e alinhar sinergias baseadas em princípios científicos.

## 🛠️ Tecnologias
- **Node.js**: Runtime assíncrono.
- **Blessed**: Layout e componentes TUI.
- **Chalk**: Estética visual em cores.
- **Math.js**: Motor de desafios matemáticos.

## 📜 Pilares (v0.9.0)
1. **Ecos da Razão**: Itens e locais referenciam Newton, Lovelace, Euler, Turing, entre outros.
2. **Intelecto como Arma**: Puzzles complexos que recompensam o conhecimento lógico.
3. **Árvore de Passivas (Legado)**: Evolução baseada em teorias científicas (Cálculo, Termodinâmica, etc).
4. **Hardcore**: Morte permanente. Sua jornada termina quando seu HP chega a zero.

## ⌨️ Controles
- **WASD**: Movimentação.
- **Q / E**: Navegação em menus.
- **F / Espaço**: Seleção.
- **V / ESC**: Voltar / Fugir.
- **I**: Inventário.
- **H**: Atributos.
- **K**: Árvore de Passivas.
- **P**: Proficiências.

## 🚀 Como Jogar
```bash
npm install
npm start
```
